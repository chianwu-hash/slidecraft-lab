## Slide 2

### Section Label

一、依據與目的

### Main Message

本年度財產盤點與檢查作業，係依上級函示、財產管理規定及本校既有作業模式辦理，目的是讓帳籍、實物與管理責任都能回到一致、清楚、可追蹤的狀態。

### Basis Card

- 115年5月2日新北府財產字第1150853771號函
- 新北市政府各機關學校經管公有財產管理檢核暨輔導計畫
- 新北市市有財產產籍管理要點第21點
- 本校114年度財產檢查暨盤點（清查）計畫及115年度簽辦資料

### Purpose Card

- 掌握財產保管、維護及使用情形
- 查核帳籍與實物是否相符
- 發現閒置、遺失、損壞或不堪使用情形
- 落實各單位財產管理責任
- 作為更正、移撥、報廢及改善依據

## Slide 3

### Section Label

二、適用範圍與盤點類別

### Main Message

本準則適用於本校各處室、專科教室及其他使用或保管校有財產之單位，盤點工作不是單一處室任務，而是全校共同配合的管理作業。

### Scope Card

- 各處室
- 專科教室
- 其他使用或保管校有財產之單位

### Property Type Cards

- 不動產：
  土地、土地改良物、房屋建築及設備
- 動產：
  機械及設備、交通及運輸設備、雜項設備
- 非消耗品：
  其他列帳管理之財物
- 有價證券及權利：
  依帳載資料核對文件與登載內容

## Slide 4

### Section Label

三、辦理原則

### Main Message

盤點與檢查不只是例行點收，而是依計畫執行、依清冊核對、依結果改善的完整管理流程。

### Principle Cards

- 每年至少一次盤點與一次檢查
- 盤點前須先完成計畫簽核
- 逐項核對帳、物、保管人與存置地點
- 檢查時同步查核新增、保管、維護、交接、報廢及報損
- 盤點與檢查後都要作成結果報告
- 發現缺失或異常時應限期改善

## Slide 5

### Section Label

四、權責分工

### Main Message

財產盤點工作要順利完成，關鍵在於總務處統籌、各單位落實自檢、檢查小組實地複查，以及教務處依既有分工會同檢查。

### Role Cards

- 總務處：
  擬訂計畫、準備清冊與表件、通知各單位、彙整自檢結果、辦理複查、製作報告、完成系統與帳務處理
- 各處室及財產保管人：
  逐項盤點、自檢、核對保管資訊、標示異常、回送資料、配合複查與改善
- 檢查小組：
  依核定計畫實地檢查、核對結果、記錄缺失、提出改善建議
- 教務處：
  依本校113年度財產檢查暨盤點（清查）計畫所列既有作業分工，會同辦理對總務處所管理財產業務之檢查

## Slide 6

### Section Label

五、115年度作業時程

### Main Message

115 年度作業從 3 月籌備開始，9 月起進入彙整與改善追蹤階段，各單位需依節點完成自檢與回送。

### Timeline

- 3月：籌備期 / 準備清冊與通知
- 4月起：盤點啟動 / 發送清冊
- 4月底前：送發完畢 / 交保管人
- 7月底前：自檢完成 / 回送總務處
- 8月底前：實地複查 / 檢查小組查核
- 9月起：結果彙整 / 改善追蹤

## Slide 7

### Section Label

六、盤點（清查）程序

### Main Message

盤點程序的核心，是依清冊逐項核對、由保管人確認、由總務處彙整並追蹤改善。

### Steps

1. 核定計畫
2. 列印清冊
3. 逐項清點
4. 保管人核章
5. 回報結果
6. 彙整報告
7. 列管追蹤

## Slide 8

### Section Label

七、財產檢查程序

### Main Message

財產檢查是在各單位自檢基礎上，進一步透過實地複查確認管理作業是否確實、程序是否合規。

### Procedure Steps

1. 核定檢查計畫
2. 實地複查
3. 查核新增與移動
4. 查核保管與維護
5. 查核標籤與存置
6. 作成檢查報告
7. 通知限期改善

### Check Focus

- 新增與移動
- 保管與維護
- 財產標籤
- 保管人與存置
- 程序查核

## Slide 9

### Section Label

八、異常處理與報廢流程

### Main Message

盤點與檢查的重點，不只是找出異常，更要依規定分流處理；其中報廢必須另案辦理，不能只靠盤點註記取代程序。

### Exception Cards

- 帳物不符：查明補正
- 盤盈：補辦登帳
- 盤虧：釐清減損
- 閒置財產：調撥或報廢
- 損壞不堪用：報廢或修復
- 保管資訊不符：立即更正

### Disposal Flow

1. 盤點標示
2. 填表附照
3. 批次送件
4. 辦理減損
5. 妥善保管

### Key Reminder

盤點註記不等於直接報廢，報廢仍須依既有作業流程另案辦理。

## Slide 10

### Section Label

九、成果文件與執行提醒

### Main Message

盤點與檢查作業完成後，除了結果回報，更要完整留存相關文件，並依後續改善與最新函示持續調整。

### Records List

- 盤點與檢查計畫
- 各單位財產清冊
- 檢查與報告表
- 盤點與追蹤紀錄
- 報廢申請與附件

### Closing Reminder

- 依核定時程辦理
- 完成自檢與複查
- 依最新規定辦理

### Closing Line

讓帳籍清楚、實物相符、責任到位，才能讓財產管理更完整、更安心。
