# AI 行政工作台品牌版 ChatGPT 生圖貼上順序

使用方式：

1. 在正確 Pro 帳號開新的 ChatGPT 生圖對話。
2. 先手動上傳頂溪國小 logo / 校徽參考圖。
3. 若有校園精靈或其他品牌角色參考圖，也在同一個對話上傳；若沒有，後續 prompt 會要求不要自行創作角色。
4. 先貼 `00`，等 ChatGPT 確認品牌與風格規則。
5. 接著依序貼 `01` 到 `15`。
6. 每張圖產生後下載，依序命名為 `01-cover.png`、`02-why-workbench.png` 等。

## 00 Brand Primer

```text
接下來我要製作一組 16:9 橫式簡報投影片圖片，主題是「AI 行政工作台」，對象是台灣國小校內行政團隊與採購審查者。

這是一份正式掛名「新北市永和區頂溪國民小學／頂溪國小」的校內提案簡報。

請依照我已上傳的頂溪國小 logo / 校徽參考圖，建立全系列品牌識別規則：

- logo / 校徽只能依照已上傳參考圖呈現，不可自行改色、改形、簡化或創造新的校徽
- 每張投影片右上角或右下角可放小型 logo / 校徽識別章，尺寸約投影片高度 6% 到 9%
- 封面與最後結語頁可以讓 logo / 校徽略大，但不可遮蓋標題
- 每張投影片請出現校名「頂溪國小」或「新北市永和區頂溪國民小學」，但不要過度重複造成擁擠
- 如果我有上傳校園精靈或其他品牌角色參考圖，才可以依照參考圖作為角落輔助元素；如果沒有上傳角色參考圖，請不要自行創作吉祥物或角色
- 不要加入英文校名、錯誤校名、虛構標語、水印、品牌標誌或外部公司 logo

全系列頁碼規則：

- 不要在圖片中生成頁碼徽章；頁碼會由後製統一疊加在左下角
- 後製格式為「01/15」「02/15」一路到「15/15」
- 頁碼徽章要小而清楚，位置和風格每張一致

全系列視覺風格：

- 溫暖、清楚、可信、務實、有制度感的國小行政提案簡報風格
- 背景以米白紙感、淺暖灰、深綠、墨黑、柔金色、少量行政藍為主
- 使用乾淨的資訊圖表、資料夾結構、行事曆、檢核表、公文資料夾、Codex 桌面版工作區、瀏覽器工作台、檔案樹與對話協作面板意象
- 視覺要像校內正式行政簡報，不要像企業科技廣告
- 避免科幻感、霓虹感、過多程式碼、密密麻麻的小字
- 如果有文字，請使用繁體中文，字要大、清楚、少量
- 不要出現真實個資、帳號、token、日曆 ID、公文號
- 每張圖都要是完整 16:9 投影片頁面，可以直接作為簡報圖片使用

請先確認你會在後續每張圖中延續這個品牌識別、頁碼規則與視覺風格。不要先產生圖片。
```
